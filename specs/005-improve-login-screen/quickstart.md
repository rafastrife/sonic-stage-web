# Quickstart: Improve Login Screen

## Setup Instructions

This feature is entirely frontend-based in the `sonic-stage-web` repository.

1. Navigate to the frontend directory:
   ```bash
   cd sonic-stage-web
   ```

2. Start the Angular development server:
   ```bash
   npm start
   ```

3. Open a browser and navigate to the application's login route (usually `http://localhost:4200/login`).

## Verifying the Design

- Confirm the dark background and centered login card.
- Confirm the presence of the neon cyan and pink accents.
- Verify that the layout is responsive by resizing the browser window (desktop, tablet, mobile views).
- Test focus states by tabbing through the input fields.
